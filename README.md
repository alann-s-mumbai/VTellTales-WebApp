# VTellTales-WebApp

A short description: Web application for VTellTales — frontend, admin and backend components.
  
## Quick start
1. Install dependencies: `npm install` (where applicable)
2. Run dev servers: see `frontend/` and `admin/` README sections
3. Build: `npm run build` or project-specific scripts

## Structure
- `frontend/` — user-facing web app
- `admin/` — admin dashboard
- `backend/` — API and services
